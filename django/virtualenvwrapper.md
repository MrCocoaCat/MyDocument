### 安装virtualenvwrapper

```
pip install virtualenvwrapper
```
### 基本使用

1. 创建虚拟环境
```
mkvirtualenv my_env
```

2. 切换到某个虚拟环境

```
workon my_env
```

3. 退出当前虚拟环境

```
deactivate
```

4. 删除虚拟环境

```
rmvirtualenv my_env
```

5. 列出虚拟环境

```
lsvirtualenv
```

6. 进入虚拟环境目录

```
cdvirtualenv
```
